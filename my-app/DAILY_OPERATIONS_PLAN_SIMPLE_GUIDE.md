## ğŸ¯ Key Concepts

### **Time Blocks**
The day is divided into 3 blocks:
- **Morning:** 05:00 - 12:00
- **Afternoon:** 12:00 - 17:00
- **Evening:** 17:00 - 22:00

Each block is independent - you can expand/collapse them.

### **Checklist Items**
These track readiness:
- â˜‘ = Confirmed (ready)
- â˜ = Pending (needs attention)

Each item has an "owner" (role, not person):
- Operations Manager
- Kitchen Manager
- Security Head
- etc.

### **Status**
- **Draft:** Still editing
- **Finalized:** Locked, ready
- **Published:** Active, executing

### **Permissions**
- **Admin/Ops Manager:** Can edit everything
- **Others:** Can only view (read-only)

## â“ Quick FAQ

**Q: Where do the rituals come from?**
A: They're automatically pulled from the Seva Planning system. You just see them here.

**Q: What if I see pending items?**
A: Try to confirm them before finalizing. If you can't, you can still finalize (it just warns you).

**Q: Can I edit after finalizing?**
A: No. Once finalized, it's locked. Plan before finalizing!

**Q: What's the difference between notes and special instructions?**
A: Notes are per time block. Special instructions are for the whole day and more visible.

**Q: Can operations staff edit?**
A: No, they can only view the finalized plan.
