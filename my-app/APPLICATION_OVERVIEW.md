# Namaha - Temple Management System
## Comprehensive Application Overview

---

## **Application Purpose & Domain**

**Namaha** is a comprehensive **Temple Management SaaS Platform** designed to manage all aspects of temple operations, from daily rituals and seva bookings to financial management, people management, and project coordination. It serves as a centralized administrative dashboard for temple administrators, staff, and management teams.

---

The main dashboard provides a comprehensive overview:

   - Multi-calendar view (Temple Calendar, Shri Gurugal, Temple Executive)
   - Daily event scheduling
   - Ritual and seva planning
   - Event load indicators (High/Medium/Low)

2. **Events Snapshot**
   - Today's events with time and type
   - Upcoming events and festivals
   - Event load visualization

3. **Task Management**
   - Task statistics (total, pending, active, done, overdue)

4. **Financial Summary**
   - Daily, monthly, yearly financial metrics

   - Expense category breakdown
   - Trend indicators

5. **Alerts & Notifications**
   - Critical alerts (e.g., priest unavailability)

6. **Announcements**

## **UI Design System**

### **Color Palette**

- **Primary Light**: `#dab76e` (Light amber/gold)

- **Primary Dark**: `#433117` (Very dark brown)
- **Primary Darkest**: `#270100` (Almost black brown)

#### **Background Colors**
- **Base**: White (`#ffffff`)
- **Subtle**: `#e7dcb9` (Lightest cream/beige)

#### **Text Colors**
- **Primary Text**: `#270100` (Darkest brown)
- **Secondary Text**: `#433117` (Very dark brown)

- **Background**: `#270100` (Dark amber)

- **Warning**: Amber (`#f59e0b`)

### **Typography**

#### **Font Families**
- **Primary (Serif)**: Noto Serif - Used for titles, headers, and logo
  - Weights: 400 (Regular), 500 (Medium), 700 (Bold)
- **Body (Sans-serif)**: Inter - Used for body text, navigation, and UI elements
  - Weights: 400 (Regular), 500 (Medium), 600 (Semi-bold)

#### **Typography Scale**
- **Page Title**: 28px, Noto Serif Bold 700
- **Section Header**: 20px, Noto Serif Medium 500
- **Body**: 16px, Inter Regular 400
- **Body Small**: 14px, Inter Regular 400
- **Body Extra Small**: 12px, Inter Regular 400
- **KPI**: 16px, Inter Semi-bold 600
- **Link**: 14px, Inter Medium 500
- **Navigation**: 17px, Inter Regular 400
- **Logo**: 20px, Noto Serif Bold 700
- **List Item**: 16px, Inter Medium 500

### **Spacing System**
Consistent spacing scale for padding, margins, and gaps throughout the application.

### **Borders & Shadows**
- Light beige borders (`#e4cc96`)
- Subtle shadows for depth and elevation
- Rounded corners (typically 2xl/3xl for modern look)

### **Animations & Transitions**
- Smooth fade-in animations
- Hover effects (lift, scale, opacity changes)
- Transition animations for state changes
- Modal enter/exit animations
- Button hover and active states

### **Component Design Principles**
- **Rounded Corners**: Extensive use of rounded-2xl and rounded-3xl
- **Card-based Layout**: White cards with subtle borders and shadows
- **Grid System**: 12-column grid layout for responsive design
- **Sticky Header**: Dark header that stays fixed at top
- **Mega Menu**: Dropdown navigation with category-based organization
- **Modal System**: Centered modals with backdrop overlays
- **Carousel Components**: Auto-rotating carousels for dashboard views

3. **Financial Management**

   - Create event → Plan logistics → Accept donations → Execute event

## **Application Philosophy**

Namaha is designed to bring **modern technology** to **traditional temple management**, providing:
- **Comprehensive Coverage**: All aspects of temple operations in one platform
- **User-Friendly Interface**: Intuitive navigation and clear visual hierarchy
- **Cultural Sensitivity**: Design that respects traditional aesthetics while being modern
- **Efficiency**: Streamlined workflows for common temple management tasks
- **Transparency**: Clear financial tracking and reporting
- **Scalability**: Support for multiple temples, branches, and hierarchies

---

## **Visual Identity**

The application's visual identity reflects:
- **Traditional Roots**: Amber/gold color palette reminiscent of temple architecture
- **Modern Functionality**: Clean, contemporary UI patterns
- **Professional Aesthetics**: Serif fonts for authority, sans-serif for readability
- **Warmth & Accessibility**: Cream backgrounds and thoughtful color contrast
- **Spiritual Respect**: Design that honors the sacred nature of temple operations
